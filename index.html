How to Build

  • Requirement
    • libclang (like libclang-dev on Ubuntu20.04) for building cppast
  • Build
> mkdir build
> cd build
> cmake .. -DCMAKE_BUILD_TYPE=Release
> make
> make install
> ls ../release/bin/
